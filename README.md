# TC Dark

Theme TC VSCode
<img src="./assets/example.png"/>

## Install
```shell
    $ code --install-extension tc-dark-0.0.1.vsix
```

## Modifying
Change colors on [theme](themes/tc-dark-color-theme.json) and do:
```shell
    $ yarn generate
```
